# 获取源码

1. 鼠标焦点移到MobaXterm

2. 在MobaXterm中输入：
    ```
    cd /home/bearpi
    ```
    然后回车

3. 在MobaXterm中输入： 
    ```
    mkdir project && cd project
    ```
    然后回车

4. 在MobaXterm中输入以下命令获取源码：
    ```
    git clone https://gitee.com/bearpi/bearpi-hm_micro_small.git
    ```

    然后回车，等待1-3分钟（根据不同网速）
