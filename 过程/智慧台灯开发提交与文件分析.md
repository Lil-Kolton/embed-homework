## 2. 模块1：E53_SC1 驱动改造 - PWM 调光

**提交**：`d1225f3ee` feat(E53_SC1): 智慧台灯驱动改造 - PWM调光/按键中断/串口日志

**修改文件**：
| 文件 | 操作 | 说明 |
|------|------|------|
| `device/st/drivers/e53_driver/E53_SC1/E53_SC1_hdf.c` | 修改 | HDF 层新增 `SetBrightness` 命令（cmd=4） |
| `device/st/drivers/e53_driver/E53_SC1/inc/E53_SC1.h` | 修改 | 新增 `E53_SC1_CMD_SET_BRIGHTNESS` 枚举 |
| `device/st/drivers/e53_driver/E53_SC1/src/E53_SC1.c` | 修改 | 新增 PWM 亮度调节逻辑 |

**新增功能**：
- PWM 亮度调节 0~100%
- 按键 GPIO 中断（E53_IO_6）切换灯光
- 串口操作日志打印
- HDF JSON 返回增加 `Brightness` 字段

**代码实现**：
- `E53_SC1.c`：通过 `E53_PWMSet()` 设置 PWM 周期和占空比，将 0~100% 亮度映射到 PWM duty
- `E53_SC1_hdf.c`：`Dispatch` 函数新增 `cmd=4` 分支，接收亮度参数并调用 `E53_SC1_SetBrightness()`

**根因**：初始 PWM period 设为 `1000000`（纳秒级），但 STM32MP1 PWM HDF 驱动将 period/duty 直接写入 TIM 计数器，`1000000` 超过 16 位 TIM 最大值，导致溢出。

- 单击：延迟 400ms 后切换开/关灯
- 长按：持续增亮
- 双击+第二次长按：持续减暗

- 初始化幂等化

**代码实现**：
```c
// 按键轮询线程
while (1) {
    int current = GpioRead(98);  // 读取 GPIO 状态
    // 消抖 + 状态机：检测按下/释放/双击/长按
    // 长按：每 100ms 亮度 ±5
    // 单击：延迟 400ms 后切换

}
```

**根因**：`Stm32Mp157GpioSetIrq` 的回调函数被忽略（`(void)func`），GPIO IRQ 不可靠。

**修改文件**：

|------|------|
| `build/lite/components/applications.json` | 注册 MQTT 组件到构建系统 |
| `tools/download_img/kernel_v2/*` | 刷新三份镜像 |

**新增功能**：
- `/bin/huawei_mqtt_demo` 可执行文件
- MQTT 3.1.1 手写报文：CONNECT / SUBSCRIBE / PUBLISH / PINGREQ / PINGRESP
- 周期上报 `LightService`（light / led_status / led_brightness）
- 订阅云端 `properties/set` topic
- 解析 `led_status` 和 `led_brightness` 下发命令
- 回复 `properties/set/response` 避免平台超时
- `Brightness > 0` 时同步上报 `led_status = true`
- 密码运行时传入，不硬编码入库

**代码实现**：
```c
// MQTT 连接
SendMqttConnect(fd);           // CONNECT + CONNACK
SendMqttSubscribe(fd);         // SUBSCRIBE properties/set/#

// 主循环
while (1) {
    select(fd, &readSet, ...); // 等待下发
    HandleMqttPacket(fd);      // 处理下发消息
    E53Sc1Read(&lamp);         // 读取传感器
    SendPropertyReport(fd, &lamp);  // 上报属性
    SendPingReq(fd);           // 保活
}

// 下发处理
HandlePropertySet(fd, topic, payload) {
    E53Sc1SetBrightness(brightness);  // 设置亮度
    E53Sc1SetLight(ledStatus);        // 设置开关
    SendSetResponse(fd, requestId, 0); // 回复平台
}
```
