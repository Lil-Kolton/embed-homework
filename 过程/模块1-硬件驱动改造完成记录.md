# 模块1 — 硬件驱动改造完成记录

> 作者：Kolton
> 日期：2026-06-11

---

## 一、做了什么？

将 E53_SC1 智慧路灯的硬件驱动升级为智慧台灯驱动，新增了 PWM 调光、按键控制和串口日志功能。

---

## 二、修改了哪些文件？

### 2.1 头文件：`E53_SC1.h`

**路径**：`device/st/drivers/e53_driver/E53_SC1/inc/E53_SC1.h`

**新增内容**：
```c
/* 按键引脚定义 */
#define E53_SC1_BUTTON_GPIO     E53_IO_6

/* PWM 参数 */
#define E53_SC1_PWM_NUM         0           // PWM 设备号
#define E53_SC1_PWM_PERIOD      1000000     // PWM 周期 1ms (单位：纳秒)
#define E53_SC1_PWM_MAX_DUTY    1000000     // 最大占空比

/* 亮度范围 */
#define E53_SC1_BRIGHTNESS_MAX  100
#define E53_SC1_BRIGHTNESS_MIN  0

/* 新增函数声明 */
int E53_SC1SetBrightness(uint8_t brightness);   // 设置亮度
uint8_t E53_SC1GetBrightness(void);             // 获取当前亮度
int E53_SC1ButtonInit(void);                     // 初始化按键
```

### 2.2 驱动实现：`E53_SC1.c`

**路径**：`device/st/drivers/e53_driver/E53_SC1/src/E53_SC1.c`

**新增内容**：

1. **全局变量**：
   - `g_brightness` — 当前亮度值（0-100）
   - `g_lightStatus` — 灯光状态（0=OFF, 1=ON）

2. **按键中断回调函数** `ButtonIrqCallback()`：
   - 按下按键时切换灯光状态
   - 开灯时默认亮度 80%
   - 每次操作打印串口日志

3. **PWM 调光函数** `E53_SC1SetBrightness()`：
   - 输入 0-100 的亮度值
   - 映射到 PWM 占空比（0-1000000ns）
   - 自动更新灯光状态

4. **按键初始化函数** `E53_SC1ButtonInit()`：
   - 配置 E53_IO_6 为上拉输入模式
   - 设置下降沿中断触发
   - 启动中断监听

5. **修改 `E53_SC1Init()`**：
   - 新增 PWM 初始化（打开设备、设置周期、启动）
   - 新增按键初始化调用
   - 新增串口日志

6. **修改 `E53_SC1DeInit()`**：
   - 新增 PWM 停止和关闭
   - 新增按键中断取消和 GPIO 释放
   - 新增串口日志

7. **修改 `E53_SC1LightStatusSet()`**：
   - ON 时调用 PWM 设置亮度（默认 80%）
   - OFF 时调用 PWM 设置亮度为 0
   - 新增串口日志

### 2.3 HDF 驱动层：`E53_SC1_hdf.c`

**路径**：`device/st/drivers/e53_driver/E53_SC1/E53_SC1_hdf.c`

**新增内容**：

1. **新增命令码**：
   ```c
   E53_SC1_SetBrightness = 4,  // 亮度设置命令
   ```

2. **扩展 JSON 返回格式**：
   - 旧格式：`{"Lux":123.45,"LED":"ON"}`
   - 新格式：`{"Lux":123.45,"LED":"ON","Brightness":80}`

3. **新增命令处理** `E53_SC1_SetBrightness`：
   - 接收字符串格式的亮度值（如 "80"）
   - 转换为整数并校验范围（0-100）
   - 调用 `E53_SC1SetBrightness()` 设置亮度
   - 返回包含亮度的 JSON 数据

## 四、命令对照表

| 命令码 | 名称 | 说明 | 数据格式 |
|--------|------|------|----------|
| 0 | Start | 初始化设备 | 无 |
| 1 | Stop | 关闭设备 | 无 |
| 2 | Read | 读取光照和状态 | 无 |
| 3 | SetLight | 开/关灯 | "ON" 或 "OFF" |
| 4 | SetBrightness | 设置亮度 | "0" ~ "100" |

## 六、注意事项

1. **PWM 引脚**：当前使用 PWM 设备号 0，实际引脚需根据硬件确认
2. **按键引脚**：使用 E53_IO_6，与 I2C（E53_IO_7/8）不冲突
3. **默认亮度**：开灯时默认 80%，可通过 SetBrightness 命令调整
4. **JSON 缓冲区**：从 100 字节扩大到 150 字节，防止 JSON 溢出
