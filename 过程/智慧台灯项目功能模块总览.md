# 智慧台灯项目功能模块总览

> 基于 BearPi-HM Micro + E53_SC1 扩展板，运行 OpenHarmony LiteOS-A 系统。
> 本文档记录从第一次提交到现在完成的全部独立功能模块。

---

## 模块1：E53_SC1 驱动改造与 PWM 调光

**目标**：将 E53_SC1 扩展板从固定亮度控制改为 PWM 无级调光（0~100%）。

**实现内容**：
- 修改 E53_SC1 HDF 驱动，新增 `E53_SC1_CMD_SET_BRIGHTNESS` 命令（cmd=4）
- 将 PWM period 修正为 TIM 计数器值 `1000`（原来误用纳秒级 `1000000` 导致溢出）
- 实现 0~100% 亮度到 PWM duty 的映射
- HDF 层 JSON 返回增加 `Brightness` 字段
- 移除 PWM 调试刷屏日志 `E53_Log("here")`
- 串口输出改为 `Brightness: 80%` 格式

**涉及文件**：
| 文件 | 说明 |
|------|------|
| `device/st/drivers/e53_driver/E53_SC1/src/E53_SC1.c` | 主驱动逻辑 |

| `device/st/drivers/e53_driver/E53_SC1/E53_SC1_hdf.c` | HDF 服务接口 |

## 模块2：本地按键控制

**目标**：使用 BearPi 开发板 F1 按键（USER_KEY1 / PG2 / GPIO98）控制台灯。

**实现内容**：
- 单击：延迟 400ms 后切换开/关灯
- 长按：持续增亮
- 双击+第二次长按：持续减暗
- 亮度最低允许 0%
- 改用 OSAL 线程轮询 `GpioRead(98)` 替代 GPIO IRQ（IRQ 中回调函数被忽略，不可靠）
- 初始化幂等化，HDF `.Init` 自动启动硬件
- 补充中文注释

**涉及文件**：
| 文件 | 说明 |
|------|------|
| `device/st/drivers/e53_driver/E53_SC1/src/E53_SC1.c` | 按键轮询状态机、调光逻辑 |
| `device/st/drivers/e53_driver/E53_SC1/inc/E53_SC1.h` | GPIO 定义、状态常量 |
| `device/st/drivers/e53_driver/E53_SC1/E53_SC1_hdf.c` | HDF `.Init` 自动初始化 |

## 模块5：HAP 滑块方向修复

**目标**：修复 Lite slider 默认 reverse=true 导致滑块方向反转的问题。

**实现内容**：
- 发现 Lite slider 组件默认 `reverse=true`，且 `reverse="false"` 属性无效
- 采用双变量反向抵消方案：
  - `brightness`（业务值，正向）→ 文字显示 + 发送命令
  - `brightnessSlider`（视觉值，反向）→ slider 绑定
  - 映射：`brightnessSlider = 100 - brightness`
- 同步处理定时滑块：`timerSliderValue = 3610 - timerDelay`
- 按钮和拖动双向同步

**涉及文件**：

## 模块8：IoTDA 下发响应与状态同步

**目标**：修复云端下发后设备不响应、灯状态不同步的问题。

**实现内容**：
- 收到 `properties/set` 后，从 topic 解析 `request_id`
- 处理完成后回复 `properties/set/response/request_id=xxx`
- 修复云端"5 分钟未响应"导致下发被阻止的问题
- 修复 `led_status` 不同步：
  - `Brightness > 0` 时上报 `led_status = true`
  - `Brightness = 0` 时上报 `led_status = false`
  - 下发亮度时同步调用 `SetLight(ON/OFF)`
- 鉴权失败排查：`CONNACK` 返回码 `0x04` 表示密码错误

**涉及文件**：
| 文件 | 说明 |
|------|------|
| `applications/.../samples/huawei_mqtt_demo/huawei_mqtt_demo.c` | set response、状态同步 |

## 最终运行方式

```sh
# 1. 连接 WiFi
cd /bin
./wpa_supplicant -i wlan0 -c /etc/wpa_supplicant.conf &

# 2. 启动 MQTT（使用 IP + 密码）
./huawei_mqtt_demo 123.60.224.23 <password>
```
